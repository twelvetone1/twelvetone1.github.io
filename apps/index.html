Version 1.6 (3 April 2022)

Minimum SDK: 27

Install Canasta scoring app

Instructions for installing this app:

  • Click on the link above to download the Android application.
  • Open the downloaded file (a pop-up will probably appear to let you do this).
  • When asked "Do you want to install" in a pop-up, click on INSTALL.
  • A "Blocked by Play Protect" pop-up might occur. If so, click on INSTALL ANYWAY.
  • If a "Send app for scanning?" pop-up appears, click on DON'T SEND.
